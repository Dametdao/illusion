# Illusion
Illusion is the offical CW Unblocker

## Features
v2 coming soon. adds **SO** much more
- Webproxy - UV
- Games - Gfiles
- Options - Basic Cloaking
- Themes - Dark And Light

## Installation
```bash
$ git clone https://github.com/chromium-workshop/illusion
$ cd illusion
$ npm install
$ npm start # Port Started On 5000
```
